路由

Hertz MVC提供了灵活强大的路由系统，支持自动路由和手动路由配置。

自动路由

使用 AutoRouters 方法可以自动注册控制器的所有公开方法：

func main() {
    app := mvc.HertzApp
    
    // 创建控制器实例
    homeController := &HomeController{}
    userController := &UserController{}
    
    // 自动注册路由
    app.AutoRouters(homeController, userController)
    
    app.Run(":8888")
}

手动路由

手动注册路由可以提供更精确的控制：

func main() {
    app := mvc.HertzApp
    
    homeController := &HomeController{}
    
    // 手动注册路由
    app.Router("/", homeController, "GET:GetIndex")
    app.Router("/about", homeController, "GET:GetAbout")
    app.Router("/contact", homeController, "POST:PostContact")
    
    app.Run(":8888")
}

Namespace路由

类似Beego的Namespace功能，可以创建结构化的路由组：

func main() {
    app := mvc.HertzApp
    
    // 创建API命名空间
    nsApi := mvc.NewNamespace("/api",
        // 自动路由
        mvc.NSAutoRouter(homeController),
        
        // 手动路由
        mvc.NSRouter("/auth/login", userController, "POST:PostLogin"),
        mvc.NSRouter("/auth/logout", userController, "POST:PostLogout"),
    )
    
    // 添加命名空间到应用
    mvc.AddNamespace(nsApi)
    
    app.Run(":8888")
}

路由参数

支持路径参数和查询参数：

// 在路由中定义参数
app.Router("/user/:id", userController, "GET:GetShow")

// 在控制器中获取参数
func (c *UserController) GetShow() {
    id := c.GetParam("id")
    page := c.GetQuery("page")
    
    c.SetData("UserID", id)
    c.SetData("Page", page)
    c.RenderHTML("user/show.html")
}
路由最佳实践
  • 使用自动路由来提高开发效率
  • 合理使用命名空间来组织API结构
  • 遵循RESTful设计原则
  • 保持路由结构清晰和一致

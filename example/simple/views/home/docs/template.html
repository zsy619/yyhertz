模板引擎

Hertz MVC使用Go原生的html/template包作为模板引擎，支持模板继承、组件化和丰富的模板函数。

基本使用

在控制器中设置数据并渲染模板：

func (c *HomeController) GetIndex() {
    // 设置模板数据
    c.SetData("Title", "首页")
    c.SetData("Message", "欢迎使用Hertz MVC")
    c.SetData("User", map[string]interface{}{
        "Name":  "张三",
        "Email": "zhangsan@example.com",
    })
    
    // 渲染模板
    c.RenderHTML("home/index.html")
}

目录结构

推荐的模板目录组织方式：

views/
├── layout/

│   └── docs/
│       ├── _sidebar.html    # 侧边栏组件
│       ├── quickstart.html  # 快速开始文档
│       └── controller.html  # 控制器文档
├── user/
│   ├── index.html          # 用户列表
│   └── profile.html        # 用户资料
└── components/
    ├── navbar.html         # 导航栏组件
    └── footer.html         # 页脚组件

        {"Title": "易用性", "Description": "类似Beego体验", "Icon": "fas fa-smile"},

    }
    c.SetData("Features", features)
    
    // 渲染时会自动继承layout
    c.RenderHTML("home/index.html")
}

自定义模板函数

// 在应用启动时注册自定义函数
func setupTemplateFunctions() {
    funcs := template.FuncMap{
        // 格式化金额
        "currency": func(amount float64) string {
            return fmt.Sprintf("¥%.2f", amount)
        },
        
        // 格式化文件大小
        "filesize": func(bytes int64) string {
            if bytes < 1024 {
                return fmt.Sprintf("%d B", bytes)
            } else if bytes < 1024*1024 {
                return fmt.Sprintf("%.1f KB", float64(bytes)/1024)
            } else {
                return fmt.Sprintf("%.1f MB", float64(bytes)/(1024*1024))
            }
        },
        
        // 生成随机颜色
        "randomColor": func() string {
            colors := []string{"primary", "secondary", "success", "warning", "info", "danger"}
            return colors[rand.Intn(len(colors))]
        },
        
        // 检查权限
        "hasPermission": func(user interface{}, permission string) bool {
            if u, ok := user.(User); ok {
                return u.HasPermission(permission)
            }
            return false
        },
        
        // 计算折扣价格
        "discount": func(price float64, percent int) float64 {
            return price * (100 - float64(percent)) / 100
        },
    }
    
    // 注册到模板引擎
    mvc.SetTemplateFuncs(funcs)
}

模板最佳实践

1. 模板组织

  • 使用布局模板避免重复代码
  • 将可复用的部分提取为组件
  • 按功能模块组织模板目录
  • 使用有意义的模板名称

2. 性能优化

// 预编译模板提高性能
func init() {
    // 预加载常用模板
    mvc.PreloadTemplates([]string{
        "layout/layout.html",
        "components/navbar.html",
        "components/footer.html",
    })
}

3. 错误处理

func (c *HomeController) GetIndex() {
    // 设置默认数据，避免模板错误
    c.SetData("Title", "首页")
    c.SetData("User", nil)
    c.SetData("Messages", []interface{}{})
    
    // 安全地设置用户数据
    if user := c.GetCurrentUser(); user != nil {
        c.SetData("User", user)
    }
    
    c.RenderHTML("home/index.html")
}

快速开始

几分钟内搭建你的第一个Hertz MVC应用。

环境要求

  • Go 1.19 或更高版本
  • Git（用于下载依赖）

安装框架

使用 go mod 初始化项目并安装Hertz MVC框架：

mkdir my-hertz-app
cd my-hertz-app
go mod init my-hertz-app
go get github.com/zsy619/yyhertz

创建第一个应用

创建 main.go 文件：

package main

import (
    "github.com/zsy619/yyhertz/framework/mvc"
    "github.com/zsy619/yyhertz/framework/middleware"
)

// 定义控制器
type HomeController struct {
    mvc.BaseController
}

func (c *HomeController) GetIndex() {
    c.SetData("Title", "Hello World")
    c.SetData("Message", "欢迎使用Hertz MVC框架！")
    c.RenderHTML("home/index.html")
}

func (c *HomeController) GetAPI() {
    c.JSON(map[string]any{
        "success": true,
        "message": "API响应成功",
        "data": map[string]any{
            "framework": "Hertz MVC",
            "version":   "1.0.0",
        },
    })
}

func main() {
    // 创建应用实例
    app := mvc.HertzApp

    // 添加中间件
    app.Use(
        middleware.LoggerMiddleware(),
        middleware.RecoveryMiddleware(),
    )

    // 创建控制器实例
    homeController := &HomeController{}

    // 自动注册路由
    app.AutoRouters(homeController)

    // 启动应用
    app.Run(":8888")
}

创建视图模板

创建目录结构和HTML模板：

mkdir -p views/home

创建 views/home/index.html：

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>{{.Title}}</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <div class="container mt-5">
        <div class="row justify-content-center">
            <div class="col-md-6">
                <div class="card">
                    <div class="card-body text-center">
                        <h1 class="card-title">{{.Title}}</h1>
                        <p class="card-text">{{.Message}}</p>
                        <a href="/home/api" class="btn btn-primary">测试API</a>
                    </div>
                </div>
            </div>
        </div>
    </div>
</body>
</html>

运行应用

执行以下命令启动应用：

go run main.go

打开浏览器访问：

  • http://localhost:8888/ - 主页
  • http://localhost:8888/home/api - API接口

项目结构

推荐的项目目录结构：

my-hertz-app/
├── controllers/         # 控制器
│   ├── home_controller.go
│   └── user_controller.go
├── views/              # 视图模板
│   ├── home/
│   │   └── index.html
│   └── layout/
│       └── layout.html
├── static/             # 静态资源
│   ├── css/
│   ├── js/
│   └── images/
├── middleware/         # 自定义中间件
├── models/            # 数据模型
├── config/            # 配置文件
├── main.go            # 入口文件
└── go.mod

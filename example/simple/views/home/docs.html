快速开始

1. 创建项目
mkdir my-hertz-app
cd my-hertz-app
go mod init my-hertz-app
2. 安装依赖
go get github.com/zsy619/yyhertz
3. 创建控制器
package main

import (
    "github.com/zsy619/yyhertz/framework/mvc"
)

type HomeController struct {
    mvc.BaseController
}

func (c *HomeController) GetIndex() {
    c.SetData("Message", "Hello Hertz MVC!")
    c.RenderHTML("index.html")
}

func main() {
    app := mvc.HertzApp
    homeController := &HomeController{}
    app.Include(homeController)
    app.Run()
}
4. 运行应用
go run main.go

API参考

控制器方法
  • SetData(key, value) - 设置模板数据
  • GetData(key) - 获取模板数据
  • RenderHTML(template) - 渲染HTML模板
  • JSON(data) - 返回JSON响应
  • Redirect(url) - 重定向
请求方法
  • GetForm(key) - 获取表单参数
  • GetQuery(key) - 获取查询参数
  • GetParam(key) - 获取路径参数
  • GetHeader(key) - 获取请求头
  • GetFile(key) - 获取上传文件
